# チャット削除ユースケース仕様書

## 概要

認証済みユーザーが所有する、指定されたチャットIDに紐づくチャット本体とメッセージを削除する。

## 入力

- `chat_id`: 削除対象チャットの識別子
- `user_id`: 認証済みユーザーID

## 正常系

1. チャット本体を取得し、所有ユーザーが認証済みユーザーと一致することを確認する。
2. 指定チャットに紐づくメッセージをすべて取得する。
3. チャット本体と全メッセージを削除する。

## 使用するポート

| Protocol | 操作 | 用途 | 送出する可能性のあるエラー |
| --- | --- | --- | --- |
| `ChatDeletionRepositoryProtocol` | `delete_chat` | 所有ユーザーに紐づくチャット本体と全メッセージを削除する | `ChatNotFoundError`, `ChatDeleteError` |

## 制約

- 削除処理が途中で失敗した場合、一部のメッセージが削除済みとなる可能性がある。

## 異常系

- チャット本体またはメッセージの削除に失敗した場合は`ChatDeleteError`を送出する。
- 対象チャットが存在しない、または所有ユーザーが一致しない場合は`ChatNotFoundError`を送出する。

## API

- `DELETE /api/chats/{chat_id}`
- 成功時: `204 No Content`
