# Presentation 層

## 責務
外部との接続部としての役割を果たす
ex) HTTPエンドポイント、バッチ処理の呼び出し口

以下のことを責務とする
- リクエスト情報をユースケース関数の引数へと変換してユースケースを呼び出す
- ユースケースの戻り値をレスポンススキーマに変換する

*ビジネスロジックに関係することは一切行わない*

## 構成
### HTTPエンドポイント
- エンドポイント: router.pyに定義
- スキーマ: schema.pyに定義

## I/O定義
- 引数: Pydantic
- 戻り値: Pydantic

## エラーハンドリング
### HTTPエンドポイント
ユースケースが投げてきたエラーをHTTPエラーへと変換する
